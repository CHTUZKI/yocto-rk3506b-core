1. 产品概述

HD-RK3506B/J-CORE 核心板是万象奥科电子有限公司基于 Rockchip RK3506B/J 处理器开发的一款低功耗，高性能的嵌入式核心板，该处理器采用三核 Cortex-A7加单核 Cortex-M0架构。主频高达1.5GHz，不仅内置了 2D 硬件引擎和显示输出引擎，还集成了如 SAI、PDM、SPDIF、音频 ADC、USB2 OTG、RMII、CAN 等十分强大的工业控制通讯接口，可满足多种现场应用需求，非常适用于工业控制、无人零售、人工智能、人机交互等应用场合。

3.引脚描述

HD-RK3506B/J-CORE 核心板将 RK3506 处理器引脚复用功能按照 HD-RK3506-IOT 评估板重新进行定义，建议用户在设计底板时参考 HD-RK3506-IOT 评估板进行，以配合产品标准驱动，减少二次开发过程。

为了保证产品设计具有良好的兼容性和稳定性，用户没有使用到的引脚资源请务必悬空处理。

HD-RK3506B/J-CORE 核心板共 128 脚，引脚通过 1.0mm间距的邮票孔引出；其实物引脚图如下图所示。
